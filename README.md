
# Dart and Flutter Studies

## Overview
Repository for learning and practicing **Dart** programming language and **Flutter** framework for mobile development.

## Contents
- Dart fundamentals and syntax
- Flutter widgets and UI components
- State management patterns
- Navigation and routing
- API integration
- Project samples and exercises

## Getting Started

### Prerequisites
- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- [Dart SDK](https://dart.dev/get-dart)
- Code editor (VS Code, Android Studio, or IntelliJ)

### Installation
```bash
git clone https://github.com/brenonun3s/studies_dart_and_flutter.git
cd studies_dart_and_flutter
flutter pub get
```

### Running Projects
```bash
flutter run
```
